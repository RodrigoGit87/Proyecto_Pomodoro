# ⏳ Pomodoro Timer

¡Bienvenido al **Proyecto Pomodoro**! Una herramienta diseñada para el focus máximo en la tarea q se esté desarrollando. Limpio, minimalista y programado para hacerte la vida más fácil.

## 🚀 ¿Qué hace este Pomodoro?
Este no es solo un temporizador de cuenta atrás; es un sistema inteligente de gestión de ciclos:
- **Concentración Máxima:** Ciclos de trabajo ininterrumpidos de 25 minutos.
- **Descanso Merecido:** Al terminar, el sistema cambia automáticamente a un modo de descanso de 5 minutos.
- **Feedback Visual:** El fondo cambia de color (`lightcoral` para trabajo, `lightgreen` para descanso) para que sepas qué toca solo con mirar de reojo.
- **Contador de Sesiones:** Lleva la cuenta de cuántos bloques de trabajo has completado en el día.

## 🛠️ Lenguajes empleados:
- **Estructura:** HTML5.
- **Estilo:** CSS3 moderno.
- **Lógica:** JavaScript puro (Vanilla JS).
